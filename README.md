# HMCTS Task App

A simple task creation system for HMCTS Junior Developer Technical Test.

- Backend: FastAPI + SQLite
- Frontend: React + Vite
- Unit tests included

## Technical Test / Task Description

This project was built as part of the **HMCTS Junior Software Developer technical test**.  
The objective was to create a simple task creation system, consisting of:

- **Backend API**: to create tasks with title, description, status, and due date/time, with validation and persistence in a database.  
- **Frontend**: to allow users to submit tasks and display the confirmation/result.

Original task details can be found [here](https://github.com/hmcts/dts-developer-challenge-junior).
